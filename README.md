Weather Now 🌤️

A responsive weather web app that lets you search any city in the world and get real-time weather data — current conditions, a 7-day forecast, and a 24-hour hourly breakdown.
Built as my first HTML/CSS/JavaScript project using the Frontend Mentor challenge as a design reference.

🛠️ Built With

HTML5 — semantic markup
CSS3 — custom properties, Flexbox, CSS Grid, animations
Vanilla JavaScript — no frameworks or libraries
Open-Meteo Weather API — free, no API key required

🧠 What I Learned
This was my first project combining HTML, CSS, JavaScript, and external APIs. Key things I picked up:

How fetch() and Promises (.then() / .catch()) work

Managing UI state — showing/hiding loading, error, and data panels

Debouncing input events to avoid spamming API calls

Using AbortController to cancel stale fetch requests and prevent race conditions

CSS custom properties (variables) for a consistent design system

Flexbox and CSS Grid for responsive layouts

The difference between position: relative, absolute, sticky, and fixed

Why [hidden] { display: none !important; } is necessary when CSS uses display: flex

Keyboard accessibility with :focus-visible and ARIA attributes
